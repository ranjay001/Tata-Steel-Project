
# TATA Steel Canteen Control & E-GatePass System

A Node.js + Express backend application that powers the **TATA Steel Canteen Control System** and **E-GatePass Visitor Management System** using **Supabase (PostgreSQL)**.

---

## 🚀 Features

- Employee Attendance Management
- Visitor Gate Pass Management
- Secure Authentication
- RESTful API
- Supabase Database Integration
- Express.js Backend
- Environment Variable Support
- CORS Enabled

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Supabase
- PostgreSQL
- JavaScript
- dotenv
- bcryptjs

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/project-name.git
```

Go to project folder

```bash
cd project-name
```

Install dependencies

```bash
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file in the project root.

Example:

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
PORT=3000
```

---

## ▶️ Run the Project

Development

```bash
npm install
```

Start Server

```bash
npm start
```

The server will start on

```
http://localhost:3000
```
